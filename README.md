# Snake pour Nintendo 3DS
Ceci est mon tout premier projet de développement !

## Objectif
Créer un jeu de Snake classique jouable sur console 3DS.

## Outils utilisés
* **Langage :** C++ (ou C)
* **Librairie :** libctru / devkitPro
* **Support :** iPad (via GitHub/Code Editor)
